# LIGHT — Deployable Small Foundation Models

This repository contains a GitHub Pages/Jekyll website for the NeurIPS 2026 workshop **LIGHT**.

## Quick start

1. Push the repository to GitHub.
2. Enable GitHub Pages from the `main` branch.
3. Keep the `baseurl` in `_config.yml` aligned with the repository name.
4. Replace tentative dates and placeholders when the venue is confirmed on OpenReview.
